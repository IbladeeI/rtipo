## Игра в дурака

### Прецедент "Создать игру"
**Основной исполнитель**: Игрок

**Описание**: Игрок создает новую комнату для игры

**Предусловие**: Игрок подключен к системе и указал свое имя

**Постусловие**: Новая игра создана, игрок ожидает подключения другого игрока

#### Основной успешный сценарий

| Пользователь                    | Система                           | 
|---------------------------------|-----------------------------------|
| 1. Подключается к системе.      |                                   | 
| 2. Вводит свое имя.             | 	                              |	
| 3. Создает комнату для игры     |                                   |
|                                 | 4. Создает комнату для игры       |

#### Альтернативный сценарий "Игрок не вводит свое имя":

| Пользователь                           | Система                                                    | 
|----------------------------------------|------------------------------------------------------------|
| 1. Подключается к системе.             |                                                            | 
| 2. Не вводит свое имя.                 | 	                                                          |	
| 3. Пытается создать комнату для игры   |                                                            |
|                                        | 4. Отображается сообщение об ошибке, предлагая ввести имя  |

### Прецедент "Покинуть игру"
**Основной исполнитель**: Игрок

**Описание**: Игрок досрочно покидает текущую игру

**Предусловие**: Игрок находится в комнате для игры

**Постусловие**: Игрок покидает комнату

#### Основной успешный сценарий

| Пользователь                    | Система                                       | 
|---------------------------------|-----------------------------------------------|
| 1. Покидает игру                |                                               | 
|                                 | 2. Завершает игру                             |
|                                 | 3. Перенаправляет пользователя в главное меню |

#### Альтернативный сценарий "Создатель комнаты для игры покидает игру":

| Пользователь                    | Система                                                                          | 
|---------------------------------|----------------------------------------------------------------------------------|
| 1. Покидает игру                |                                                                                  | 
|                                 | 2. Завершает игру                                                                |
|                                 | 3. Отображает сообщение другому игроку о том, что создатель комнаты покинул игру |
|                                 | 4. Перенаправляет всех игроков в главное меню                                    |

### Прецедент "Присоединиться к игре"
**Основной исполнитель**: Игрок

**Описание**: Игрок присоединяется к существующей игре

**Предусловие**: Игрок подключен к системе и указал свое имя

**Постусловие**: Игрок присоединен к игре, ожидает начала игры

#### Основной успешный сценарий

|------------------------------------|-----------------------------------------------|

#### Альтернативный сценарий "Свободных мест нет":

| Пользователь                       | Система                                                  | 
|------------------------------------|----------------------------------------------------------|
| 1. Пытается присоединиться к игре  |                                                          |
|                                    | 2. Проверяет наличие свободного места в игре             |

### Прецедент "Выставить карту"
**Основной исполнитель**: Игрок

**Описание**:  Игрок выставляет карту на стол, начиная ход

**Предусловие**: Игрок присутствует в комнате для игры, игра запущена, ход предоставлен игроку и не выброшено ни одной карты

**Постусловие**: Игрок сделал первый ход, ожидает ответного хода от другого игрока

#### Основной успешный сценарий

| Пользователь                       | Система                                       | 
|------------------------------------|-----------------------------------------------|
| 1. Выставляет карту                |                                               |
|                                    | 2. Отображает выставленную карту на столе     |

---
### Прецедент "Подкинуть карту"
**Основной исполнитель**: Игрок

**Описание**:  Игрок подкидывает карту, значение которой уже выложено на игровом столе

**Предусловие**: Игрок присутствует в комнате для игры, игра запущена, ход предоставлен игроку, выброшена хотя бы одна карта

**Постусловие**: Игрок подкинул карту отбивающемуся

#### Основной успешный сценарий

| Пользователь                        | Система                                       | 
|-------------------------------------|-----------------------------------------------|
| 1. Пытается подкинуть карту на стол |                                               |
|                                     | 2. Проверяет количество карт у отбивающегося  |
|                                     | 3. Проверяет корректность хода                |
|                                     | 4. Отображает подкинутую карту на столе       |

#### Альтернативный сценарий "Значения подкидываемой карты нет на столе":

| Пользователь                        | Система                                       | 
|-------------------------------------|-----------------------------------------------|
| 1. Пытается подкинуть карту на стол |                                               |
|                                     | 2. Проверяет количество карт у отбивающегося  |
|                                     | 3. Проверяет корректность хода                |
|                                     | 4. Отображает сообщение о некорректном ходе   |

#### Альтернативный сценарий "У отбивающегося не осталось карт":

| Пользователь                        | Система                                       | 
|-------------------------------------|-----------------------------------------------|
| 1. Пытается подкинуть карту на стол |                                               |
|                                     | 2. Проверяет количество карт у отбивающегося  |
|                                     | 3. Проверяет количество карт в колоде         |
|                                     | 4. Отображает сообщение о некорректном ходе   |

#### Альтернативный сценарий "У отбивающегося не осталось карт и в колоде нет карт":

| Пользователь                        | Система                                       | 
|-------------------------------------|-----------------------------------------------|
| 1. Пытается подкинуть карту на стол |                                               |
|                                     | 2. Проверяет количество карт у отбивающегося  |
|                                     | 3. Проверяет количество карт в колоде         |
|                                     | 4. Объявляет победителя и завершает игру      |

#### Альтернативный сценарий "Нападющий подкинул последнюю карту и в колоде нет карт":

| Пользователь                        | Система                                       | 
|-------------------------------------|-----------------------------------------------|
| 1. Пытается подкинуть карту на стол |                                               |
|                                     | 2. Проверяет количество карт у отбивающегося  |
|                                     | 3. Проверяет корректность хода                |
|                                     | 4. Отображает подкинутую карту на столе       |
|                                     | 5. Проверяет наличие карт у нападающего       |
|                                     | 6. Проверяет наличие карт в колоде            |
|                                     | 7. Объявляет победителя и завершает игру      |

### Прецедент "Взять карты"
**Основной исполнитель**: Игрок

**Описание**:  Игрок забирает карты со стола, не побив их

**Предусловие**: Игрок присутствует в комнате для игры, игра запущена, ход предоставлен другому игроку, выброшена хотя бы одна карта

**Постусловие**: Игрок забрал карты, ход передается следующему игроку

#### Основной успешный сценарий

| Пользователь                        | Система                                       | 
|-------------------------------------|-----------------------------------------------|
| 1. Забирает карты со стола          |                                               |
|                                     | 2. Передает карты на руки игрока              |
|                                     | 3. Завершает и передает ход другому игроку    |


---
### Прецедент "Завершить ход"
**Основной исполнитель**: Игрок

**Описание**:  После отбития всех карт, игрок завершает свой ход, передавая ход следующему игроку

**Предусловие**: Игрок присутствует в комнате для игры, игра запущена, ход предоставлен игроку

**Постусловие**: Ход игрока завершен. Ход передан следующему игроку

#### Основной успешный сценарий

| Пользователь                        | Система                                       | 
|-------------------------------------|-----------------------------------------------|
| 1. Завершает ход                    |                                               |
|                                     | 2. Проверяет наличие карт в колоде            |
|                                     | 3. Проверяет наличие карт у игроков           |
|                                     | 4. Передает ход другому игроку                |

#### Альтернативный сценарий "Хотя бы у одного игрока после завершения хода нет 6 карт на руках":

| Пользователь                        | Система                                       | 
|-------------------------------------|-----------------------------------------------|
| 1. Завершает ход                    |                                               |
|                                     | 2. Проверяет наличие карт в колоде            |
|                                     | 3. Проверяет наличие карт у игроков           |
|                                     | 4. Раздает недостающие карты игрокам          |
|                                     | 5. Передает ход другому игроку                |

---

| Пользователь                        | Система                                       |

|-------------------------------------|-----------------------------------------------|
| 1. Пытается отбить карту            |                                               |
|                                     | 2. Проверяет корректность хода                |
|                                     | 3. Отображает карту, поверх отбиваемой        |
